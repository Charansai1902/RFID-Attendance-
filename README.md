🌐 RFID-Based Student Attendance Management System
An automated, end-to-end Internet of Things (IoT) and full-stack web application designed to optimize and streamline classroom attendance tracking. By substituting manual roll calls with contactless smart card validation, this system eliminates administrative overhead, mitigates proxy logging, and surfaces analytical attendance insights in real time on a centralized administrative dashboard.

🚀 System OverviewTraditional attendance methodologies are inherently prone to inefficiencies, proxy vulnerabilities, and data gaps. This project provides an elegant, physical-to-digital bridge: an ESP32 Edge Device reads the radio frequency signatures of passive student identification tags via an MFRC522 RFID Module.The captured unique identifier (UID) is immediately processed and piped over a local Wi-Fi layer directly into a Node.js runtime server. The server parses the hardware logging transmission, records timestamps, increments attendance metrics, and instantly forces updates onto a client-facing monitoring dashboard.✨ Key FeaturesReal-Time Data Streams: Automated state synchronization polls the server database every 5000ms for seamless updates without manual page refreshes.Granular Metrics Visualization: Tabulates structural info including Roll Number, Student Name, Hardware RFID UID, Total Conducted Classes (Fixed to 60), Presence Frequency, and exact Attendance Percentages.Live Context Filter Engine:

An embedded high-performance string matching listener allows administrators to instantaneously filter the roster by Name, Roll Number, or RFID Hex Strings.Classroom Analytics Display: Dynamically computes and displays the Total Students Registered, Total Number of Classes, and the Average Class Attendance Percentage.Dual Hardware/Software Validation: Offers hardware feedback loop alerts (Piezo audio chirp and active visual Green/Red LEDs) synchronized with local 20x4 I2C liquid crystal displays.🛠️ Hardware ArchitectureRequired ComponentsESP32 Wroom DEVKIT V1 – Core processing unit equipped with a native Wi-Fi TCP/IP stack.MFRC522 RFID Reader/Scanner – Operating on high-frequency 13.56 MHz electromagnetic waves to decrypt passive card transponders.20x4 Character LCD Display (with integrated I2C Serial Interface Module) – Serial output display to communicate tracking confirmations at the classroom edge.Active Piezo Buzzer – Generates a sharp audible sound signature when an RFID scan completes.Dual Status Indicator LEDs (Green & Red) –
